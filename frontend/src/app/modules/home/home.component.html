GPX Web editor

Webová aplikace pro ukládání, úpravu a sdílení souborů GPX. Projekt je určen k nasazení na Firebase. Lze je však exportovat do jednoduchého formátu jednostránkové webové stránky.

GitHub repozitář: https://github.com/stanek-r/gpx-web-editor

Editor:

V současné době je možnost nahrávat/stahovat/editovat a vytvářet jednotlivé GPX soubory a následně je zobrazit na mapě
